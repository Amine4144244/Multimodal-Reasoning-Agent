# Frontend API Integration Skill

Use this skill when connecting frontend to backend APIs.

## Requirements
- typed/structured request and response handling
- loading/error/success states
- retries only when appropriate
- graceful empty-state handling
- auth-aware request flow if needed

## Good Practices
- centralize API client logic
- normalize backend errors into UI-friendly messages
- keep transport logic separate from UI components
- handle timeouts and cancellation where useful

## Deliverables
- API client design
- request/response contract
- UI state handling
- auth/error behavior
- testing notes
