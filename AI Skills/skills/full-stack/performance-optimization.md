# Performance Optimization Skill

Use this skill when improving performance.

## Process
1. define performance issue
2. measure baseline
3. identify bottleneck
4. optimize the bottleneck
5. re-measure
6. document tradeoffs

## Areas to Inspect
- frontend bundle/render behavior
- API latency
- database queries/indexes
- caching opportunities
- AI model latency/cost
- retrieval performance
- unnecessary tool calls

## Deliverables
- bottleneck summary
- metrics before/after
- optimization changes
- tradeoffs
- follow-up ideas
