# Testing Strategy Skill

Use this skill when defining or implementing tests.

## Testing Pyramid Guidance
- unit tests for isolated logic
- integration tests for module interaction
- API tests for backend contracts
- component tests for key UI logic
- end-to-end tests for critical user journeys
- evaluation tests for AI workflows

## Prioritize Tests For
- business-critical flows
- fragile logic
- auth/security-sensitive behavior
- parsing/transformation logic
- agent state transitions
- retrieval and tool-use correctness

## Deliverables
- what to test
- test levels
- sample cases
- mocks/stubs strategy
- gaps not covered
