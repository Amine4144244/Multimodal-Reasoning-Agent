# Full-Stack Feature Implementation Skill

Use this skill when building a feature across frontend, backend, database, and/or AI layers.

## Workflow
1. define user story
2. define acceptance criteria
3. map affected layers
4. define contracts between layers
5. implement backend/domain first where appropriate
6. implement frontend integration
7. add tests
8. validate end-to-end flow

## Deliverables
- feature summary
- data flow
- files to change
- implementation steps
- test plan
- rollout risks
