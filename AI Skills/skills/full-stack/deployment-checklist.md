# Deployment Checklist Skill

Use this skill before deploying features or systems.

## Checklist
- environment variables configured
- secrets managed securely
- migrations reviewed
- logging/monitoring enabled
- error handling verified
- rate limits/security reviewed
- build passes
- tests pass
- rollback plan exists
- feature flags considered if needed

## AI-Specific Deployment Checks
- prompt versions tracked
- tool permissions reviewed
- tracing enabled
- evaluation baseline recorded
- fallback behavior tested
- cost guardrails considered

## Deliverables
- deployment readiness summary
- risks
- manual verification steps
- rollback notes
