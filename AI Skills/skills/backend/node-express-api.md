# Node.js / Express API Skill

Use this skill when implementing backend APIs.

## Principles
- keep routes thin
- put business logic in services
- validate request inputs
- return consistent response shapes
- centralize error handling

## Suggested Layers
- routes/controllers
- services
- repositories/data-access
- validation
- middleware

## API Checklist
- input validation
- auth/authorization
- error cases
- response schema
- pagination/filtering if relevant
- logging
- test coverage

## Deliverables
- endpoint definitions
- request/response schemas
- service design
- error handling strategy
- affected files
